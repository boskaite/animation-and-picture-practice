Практикум по вставке изображений и анимации на страницу

1. Картинка как фон

1.1 background-size: contain. Подстановка изображения в исходных размерах с повторениями. Выходящее за пределы обрезается.

1.2 background-size: cover. Заполнение элемента фоновым изображением по наибольшей из сторон без повторения, лишнее обрезается.

1.3 background-size: 100px. Заполнение картинками заданным размером 100px по длине изображения. Высота автоматическая, пропорциональна исходному размеру.

1.4 background-size: auto 40px. Заполнение картинками заданными размером 40рх по высоте. Длина автоматическая, пропорциональна исходному размеру.

1.5 background-size: 150px 40px. Заполнение картинками заданными размерами 150 рх по длине и 40рх по высоте. Картинка искажается.

1.6 background-repeat:no-repeat. Подстановка изображения в исходных размерах без повторения.

1.7 background-repeat:no-repeat; background-size: 150px. Подстановка изображения размером 150px по длине без повторения.
