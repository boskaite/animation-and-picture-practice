1. Картинка как фон

1.1 Размеры и повторения отображения фонового изображения

1.1.1 background-size: contain. Подстановка изображения в исходных размерах с повторениями. Выходящее за пределы обрезается.

1.1.2 background-size: cover. Заполнение элемента фоновым изображением по наибольшей из сторон без повторения, лишнее обрезается.

1.1.3 background-size: 100px. Заполнение картинками заданным размером 100px по длине изображения. Высота автоматическая, пропорциональна исходному размеру.

1.1.4 background-size: auto 40px. Заполнение картинками заданными размером 40рх по высоте. Длина автоматическая, пропорциональна исходному размеру.

1.1.5 background-size: 150px 40px. Заполнение картинками заданными размерами 150 рх по длине и 40рх по высоте. Картинка искажается.

1.1.6 background-repeat:no-repeat. Подстановка изображения в исходных размерах без повторения.

1.1.7 background-repeat:no-repeat; background-size: 150px. Подстановка изображения размером 150px по длине (ширине) без повторения.

1.1.8 background-repeat:no-repeat; background-size: auto 150px. Подстановка изображения размером 150px по высоте без повторения.

1.1.9 background-repeat:no-repeat; background-size: 300px 80px. Подстановка изображения размером 300px по длине (ширине) и 80px по высоте без повторения.

1.1.10 background-repeat:no-repeat; background-size: 100% 80px. Подстановка изображения размером 100% по длине (ширине) и 80px по высоте без повторения.

1.1.11 background-repeat:no-repeat; background-size: 100px 100%. Подстановка изображения размером 100px по длине (ширине) и 100% по высоте без повторения.

1.1.12 background-repeat:repeat-x; Подстановка изображения размером 100px по длине (ширине) c повторением только по горизонтали.

1.1.13 background-repeat:repeat-x; Подстановка изображения размером 50px по высоте c повторением только по вертикали.

1.2 Позиционирование фонового изображения

Подстановка изображения размером 150px по длине (ширине) без повторения.

1.2.1 Позиционирование поумолчанию.

Свойство background-position может принимать одно или два значения в различных единицах измерения.

1.2.2 background-position: left; Позиционирование слева блока в центре высоты.

1.2.3 background-position: center; Позиционирование в центре блока.

1.2.4 background-position: right; Позиционирование справа блока в центре высоты.

1.2.5 background-position: top; Позиционирование вверху блока по центру длины.

1.2.6 background-position: bottom; Позиционирование внизу блока по центру длины.

1.2.7 background-position: 20%; Позиционирование 20% длины, центр высоты.

1.2.8 background-position: 250px; Позиционирование 250px длины, центр высоты.

1.2.9 background-position: left bottom; Позиционирование слева снизу. Bottom может быть заменено на любое значение в %, px и прочие ед. измерения длины.

1.2.10 background-position: right top; Позиционирование справа, сверху.

1.2.11 background-position: right bottom; Позиционирование справа, снизу. Bottom может быть заменено на любое значение в %, px и прочие ед. измерения длины.

1.2.12 background-position: 20% 20%; Позиционирование 20% длины 20% высоты.

1.2.13 background-position: 250px 80px; Позиционирование 250px длины 80px высоты.

Свойство background-position-x может принимать одно значение в различных единицах измерения.

1.2.14 background-position-x: center; Позиционирование по длине - по центру.

Свойство background-position-y может принимать одно значение в различных единицах измерения.

1.2.14 background-position-y: bottom; Позиционирование по высоте - снизу.

1.2.15

Свойство background-attachment: fixed; фиксирует изображение в блоке относительно экрана пользователя не зависимо от прогрутки элементов в блоке

1.3 Несколько фоновых изображений

1.3.1 background-position-y: bottom; Первое изображение находится всегда сверху, последующие изображения по порядку накладываются под предыдущее изображение.
